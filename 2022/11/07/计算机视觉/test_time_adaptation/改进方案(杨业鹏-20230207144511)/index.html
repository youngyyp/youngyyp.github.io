改进和测试方案

  • 训练集采用CIFAR10；测试集和验证集采用CIFAR10-C

  • 先在CIFAR10数据集上进行常规训练(sgd进行更新)，且同时使用DLDR进行网络权重采样得到低维度优化轨迹子空间（40维）

      python -u train_sgd.py --datasets CIFAR10 --lr 0.1 --corrupt 0 --arch=resnet20 --epochs=150  --save-dir=save_labelnoise$c\_$model |& tee -a log_$model

  • 然后在测试集上进行无监督的test time adaptation：

    • 优化算法采用P-BFGS（投影到优化轨迹子空间的二阶优化算法）
    • 损失函数改为ENTROPY

  • 最后比较性能：

    • source
    • 原tent算法
    • 原pbfgs
    • 本方案改进的算法


python train_sgd.py --datasets CIFAR10 --lr 0.1 --corrupt 0 --arch=resnet20 --epochs=150  --save-dir=test_save_labelnoise0_resnet20 

python train_pbfgs.py --epochs 20 --datasets CIFAR10 --corrupt 0. --params_start 0 --params_end 81  --batch-size 1024   --n_components 40 --arch=resnet20  --save-dir=test_save_labelnoise0_resnet20

python train_psgd.py --epochs 40 --datasets CIFAR10 --lr 1 --corrupt 0 --params_start 0 --params_end 81  --batch-size 128  --n_components 40 --arch=resnet20  --save-dir=test_save_labelnoise0_resnet20
