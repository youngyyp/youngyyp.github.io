    x 的平方根：

    给你一个非负整数 x ，计算并返回 x 的 算术平方根 。

    由于返回类型是整数，结果只保留 整数部分 ，小数部分将被 舍去 。

    要求时间复杂度为O(logn)，空间复杂度为O(1)

    二分法，找到使mid * mid <= x的最大mid

    //标准左闭右闭的写法，就按着这个来写，不容易混；
//初始化分别等于左右边界值，每次更新边界左加右减
class Solution {
public:
    int mySqrt(int x) {
        int l = 0, r = x, ans = -1;
        while (l <= r) {
            int mid = l + (r - l) / 2;
            if ((long long)mid * mid <= x) {
                ans = mid;
                l = mid + 1;
            } else if ((long long)mid * mid > x) {
                r = mid - 1;
            } else{
                return mid;
            }
        }
        return ans;
    }
};

    // 标准左开右闭的写法，右边界初始化要加1，因为不含右边界；左边界更新时要+1,因为含左边界
class Solution {
public:
    int mySqrt(int x) {
        long long l = 0, r = (long long) x + 1, ans = -1;
        while (l < r) {
            int mid = l + (r - l) / 2;
            if ((long long)mid * mid < x) {
                ans = mid;
                l = mid + 1;
            } else if((long long)mid * mid > x){
                r = mid;
            } else return mid;
        }
        return ans;
    }
};

    // 左开右闭 我的写的很乱
class Solution {
public:
    int mySqrt(int x) {
        int left = 0;
        int right = x;
        long long mid;
        int ans;
        while(left < right - 1){
            mid = left + (right -left)/2;
            if(mid * mid > x){
                right = mid;
            }else if(mid * mid < x){
                left = mid;
                ans = mid;
            }else return mid;
        }
        return ans;
    }
};

    c++虚函数是什么：

    C++中的虚函数是一种特殊的函数，可以实现多态性。当一个类中的函数被声明为虚函数时，它将成为一个动态绑定的函数。这意味着，当该函数被调用时，实际执行的函数将取决于实例对象的类型而不是指针或引用的类型。虚函数的使用可以大大提高代码的灵活性和可维护性。

    虚函数的声明方式为在函数声明前加上关键字“virtual”，例如：

    c++Copy codeclass Animal {
public:
    virtual void makeSound() {
        std::cout << "The animal makes a sound" << std::endl;
    }
};

class Dog : public Animal {
public:
    void makeSound() {
        std::cout << "The dog barks" << std::endl;
    }
};

class Cat : public Animal {
public:
    void makeSound() {
        std::cout << "The cat meows" << std::endl;
    }
};

    在上面的例子中，Animal类中的makeSound函数被声明为虚函数，因此它将在Dog和Cat类中被重写。当我们通过指向派生类的指针调用makeSound函数时，会根据实际类型进行调用：

    c++Copy codeint main() {
    Animal* a = new Animal();
    Animal* d = new Dog();
    Animal* c = new Cat();

    a->makeSound();  // "The animal makes a sound"
    d->makeSound();  // "The dog barks"
    c->makeSound();  // "The cat meows"

    delete a;
    delete d;
    delete c;

    return 0;
}

    多进程（一个进程往队列存图片，另一个进程取图片，怎么写取图片的代码），下面是一个基本的思路：

    1. 创建两个进程，进程1和进程2，它们共享一个环形队列的内存区域。
    2. 进程1往队列中存储图片数据，进程2从队列中取出图片数据。
    3. 为了确保并发访问时数据不会出现冲突，可以使用信号量来实现进程间的同步和互斥。例如，使用一个信号量记录当前空闲槽位的数量，每次往队列中存储图片时，需要先等待有空闲的槽位，然后将图片数据写入队列，并增加已使用槽位的数量。每次从队列中取出图片时，需要先等待有已使用的槽位，然后读取队列中的图片数据，并增加空闲槽位的数量。
    4. 在进程间共享数据时，需要使用进程间通信机制，例如共享内存。为了防止数据出现竞争，需要对共享内存进行加锁和解锁操作。例如，在进程1中写入图片数据时，需要先对共享内存加锁，写入数据后再解锁。在进程2中读取图片数据时，也需要先对共享内存加锁，读取数据后再解锁。
